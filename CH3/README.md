### 단위행렬:
단위행렬은 주대각선의 원소가 모두 1이고 나머지 원소가 0인 정사각행렬이다. 어떤 행렬 A와 곱해도 A가 그대로 유지된다.

### 역행렬:
역행렬은 정사각행렬 A에 대해 A와 곱했을 때 단위행렬이 되는 행렬이다. \(A A^{-1} = I\)를 만족한다.

### 전치행렬:
전치행렬은 행렬의 행과 열을 서로 바꾼 행렬이다. 원소 \(a_{ij}\)의 위치를 \(a_{ji}\)로 바꾼 것이다.

## 파이썬 리스트와 넘파이 ndarray의 데이터 저장방식을 비교하고 넘파이가 처리속도가 빠른 이유를 설명하라

파이썬 리스트는 각 요소가 객체로 저장되어 참조 방식으로 메모리에 분산된다.  
NumPy ndarray는 같은 자료형의 데이터를 연속된 메모리 공간에 저장한다.  
그래서 NumPy는 메모리 접근이 효율적이고 벡터화 연산이 가능해 더 빠르다.
