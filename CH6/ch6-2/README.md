# 실습과제 2
- α 값이 1보다 큰 경우에는 입력 영상의 픽셀 값이 전체적으로 확대되어 밝은 영역은 더욱 밝아지고 어두운 영역은 더욱 어두워지게 된다. 이로 인해 픽셀 값 간의 차이가 커지면서 영상의 대비가 증가하고 전체적으로 선명한 영상이 된다.
- α 값이 0과 1 사이인 경우에는 입력 영상의 픽셀 값이 전체적으로 축소되어 밝기 차이가 줄어들게 된다. 이로 인해 밝은 영역과 어두운 영역의 구분이 약해져 영상이 전체적으로 흐릿해지고 대비가 감소한다.
- α 값이 0보다 작은 경우에는 픽셀 값의 부호가 반전되는 효과가 발생하여 밝은 부분과 어두운 부분의 관계가 뒤바뀌게 된다. 따라서 원본 영상과는 반대되는 밝기 분포를 가지게 되어 반전 영상 효과가 나타난다.
