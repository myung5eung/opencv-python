## 영상처리, 컴퓨터 비전기술이 자율주행, 로봇분야에서 사용되는 사례를 조사하시오
자율주행에서는 카메라 영상으로 차선을 검출하고 객체 검출 알고리즘으로 차량·보행자·신호등을 인식하여 주행 경로를 결정한다. 로봇에서는 영상 기반으로 물체 위치를 찾고 SLAM 기술로 지도를 만들며 이동한다.
## CMOS 센서의 원리를 자세히 조사하시오.
CMOS image sensor는 각 픽셀에 있는 포토다이오드가 빛을 받아 전자를 생성하고, 이 전하를 전압 신호로 변환한다. 각 픽셀에는 증폭기와 선택 스위치가 있어 행/열 단위로 신호를 순차적으로 읽어내며, 이후 아날로그 신호를 내부 A/D 변환기로 디지털 값으로 변환해 영상 데이터를 만든다.
## 아날로그신호를 디지털신호로 변환하는 과정에서 샘플링과 양자화의 차이를 설명하시오.
샘플링: 무한한 연속된 값을 일정한 해상도에 따라 유한개의 화소수만큼입력 값을 취하는 과정<br>
양자화: 제한된 비트 수로 화소값을 표현하려 밝기 값을 정수화 시키는 과정
## 영상의 해상도의 정의와 이미지 센서의 개수의 관계를 설명하시오.
영상의 해상도(화질, 정밀도)는 총 픽셀 수를 의미하여 이미지 센서의 개수에 의해서 결정된다. 이미지 센서의 개수가 많을수록 더 세밀하게 표현이 가능하기 때문에 원본에 가까운 영상을 획득할 수 있다.
