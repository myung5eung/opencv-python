# 실습과제 1
### Windows 미디어 플레이어에서 지원하는 코덱의 FOURCC 코드를 조사하시오
코덱: 동영상 파일의 용량을 줄이기 위해 영상을 압축하고, 재생할 때 다시 풀어 주는 압축·복원 방식으로 H.264, MPEG-4, Motion JPEG, Windows Media Video 등이 해당한다.
FOURCC: Four Character Code의 약자로, 코덱이나 영상 형식을 네 글자의 코드로 나타낸 것으로 MJPG, XVID, DIVX, H264, mp4v, WMV3 등이 FOURCC 코드이다.
Windows 미디어 플레이어에서 지원하는 대표적인 FOURCC 코드: MJPG, WMV1, WMV2, WMV3, WVC1, H264, AVC1, MP4V, mp4v, XVID, DIVX, DVSD, VP80, I263, MPG1, MPG2, MP2V
